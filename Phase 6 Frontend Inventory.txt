## Authentication Flow

### Current Auth Implementation
1. Login process: Mock login with hardcoded user data
2. Token storage: In-memory ApiService instance (not persisted)
3. Token refresh: Not implemented
4. Protected routes: Conditional rendering based on user state
5. Logout process: Clear user state and reset to login page

### Auth State Management
Using Zustand store to manage authentication state globally across the application.
